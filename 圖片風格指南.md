> 目標：所有上架圖長得像「同一個品牌出品的型錄」，不是一張一張各自好看。
> 風格基底：UNIQLO 的紀律 × 小草作 Grassphere 的療癒質感（Tony 指定的同類型參考）。

## 零、主要參考：小草作 Grassphere（比 UNIQLO 更近的同類品牌）

台灣花草茶品牌（shop.grassphere.com），「植感療癒」調性、賣的是日常儀式感——跟極光盾的受眾語言幾乎重疊。2026-07-07 實地看過官網＋蝦皮賣場（參考圖存 scratchpad gp_1〜4），拆出五招：

1. **一組色票，不是一個底色**：蜜桃、奶油、薄荷輪替，但飽和度、明度一致，賣場整面看還是同一家。→ 極光盾版：暖米白為主底，允許「品牌黃刷淡」「標籤藍刷淡」當副底色，做出系列感又不亂
2. **內容物攤開＝誠實敘事**：乾燥果材、花草攤在陶盤上拍——真實、可口、零 AI 味。→ 沐浴包的艾草／粗鹽／花瓣、薰香粉的粉末原料，全部適用這招，這是「AI 圖疲乏」時代最強的差異化
3. **圓形嵌入小圖**：主圖角落嵌一個圓形「使用結果」照（泡好的茶湯）。→ 極光盾版：噴霧的使用瞬間、沐浴包在浴缸水面的樣子
4. **包裝自己會講話就少疊字**：他們縮圖幾乎無字，靠包裝設計載品牌。→ 淨化露藍標籤自帶品牌＋吉祥獸，圖上文字可比一般賣家少，但保留一句黃標籤鉤子（我們的品類需要說明給誰用）
5. **道具語彙固定**：陶盤、素織品、鮮花重複出現成為品牌符號。→ 極光盾定案道具庫：木托盤、棉麻布、乾燥花、素陶皿，只用這四樣

## 一、UNIQLO 圖片風格分析（我們要學的是「方法」不是「長相」）

UNIQLO 的圖片為什麼強？拆解成 6 個可複製的原則：

### 1. 產品是唯一主角
畫面裡沒有任何東西跟產品搶眼睛。大量留白、道具極少（甚至沒有）、焦點永遠只有一個。
**行銷邏輯**：蝦皮搜尋結果頁一格只有指甲大，主角不清楚＝直接被滑過去。

### 2. 全店統一底色 → 賣場變型錄
UNIQLO 全站商品圖用同一套淺灰白底、同一種打光。單張看是「乾淨」，整頁看就變成「品牌感」——這是它最厲害的複利效果。
**行銷邏輯**：消費者點進賣場看到整齊劃一的圖牆，瞬間從「路邊攤」升級成「專櫃」的信任感。

### 3. 功能標籤化（不寫長文案）
HEATTECH、AIRism 這種小徽章＋兩三個字的功能點（保暖、涼感、快乾），從不寫一段話。資訊是「掃」的，不是「讀」的。
**行銷邏輯**：消費者在手機上停留 1 秒，標籤能傳達的，長文案傳達不了。

### 4. 情境圖也極簡
就算是模特兒穿搭圖，背景也是素色或極簡空間，情境只負責「示範用法」，不負責「講故事講到失焦」。

### 5. 字體排版有紀律
同一套字體、固定的字級層次（大標／小字兩級就夠）、固定的擺放位置。每張圖的文字像同一個人排的。

### 6. 低飽和、自然光感的色彩情緒
不用螢光色、不用重滤鏡，顏色乾淨誠實——賣的是「生活的樣子」，不是「特效」。

## 二、極光盾轉譯：「暖色版的 UNIQLO」

UNIQLO 是冷靜的都會極簡；極光盾受眾是 35-44 歲敏感體質女性，要的是**安心感**。所以方法照抄、溫度換掉：

**品牌識別（2026-07-07 Tony 定案）**：上架圖一律用「極光盾＋NORTHSHIELD」文字字標（可加一小段品牌黃橫線），**吉祥獸是草稿、不得用於上架圖**（瓶標包裝上原有的小徽章屬產品實拍、不動）。

**主圖必備用途說明（Tony 定案）**：我們的品類價值看不見（磁場拍不出來）、顧客是帶著問題來搜尋的，主圖必須一秒回答「這是什麼＋什麼時候用」——用途大標（黃標籤）＋場景副標是主圖的固定配備，跟小草作的無字路線刻意不同。

| UNIQLO 原則 | 極光盾版本 |
|---|---|
| 淺灰白統一底色 | **暖米白／奶油色統一底色**（建議 #F7F3EA 附近），全店一致 |
| 紅色 price block 當視覺錨點 | **品牌黃 #e6ca5c** 當唯一強調色（標籤、細線、小色塊） |
| 冷白均勻棚燈 | **柔和晨光感**：光要暖、影子要軟 |
| 功能標籤（保暖、快乾） | **儀式標籤**（見下方文案規則，絕不寫效果） |
| 無襯線黑體 | **Noto Sans TC**，大標＋小字兩級，位置固定 |
| 道具＝零 | 道具最多 1-2 件：乾燥花、木質托盤、棉麻布（固定這幾樣，變成品牌符號） |

### 蝦皮圖組的角色分工（一個商品一套圖，每張有工作）

### 蝦皮搜尋頁實況（2026-07-07 實地看過，Tony 指正後修正）

實際逛蝦皮搜尋結果頁：**多數賣家的主圖不是乾淨去背照**，而是「照片＋大字鉤子＋折扣角標」的設計圖，因為縮圖只有指甲大、又跟幾十張圖同場competing。純乾淨圖是少數派。

所以主圖策略修正為**混血版**：底子照 UNIQLO 紀律（統一暖底、產品佔 60%、低飽和），上面疊**一句 4〜8 字的鉤子標籤**（品牌黃圓角標籤、固定版位）＋一行小副標。既有點擊力，整個賣場看起來又還是同一套型錄。

**文字一律用程式疊（Noto Sans TC、固定字級與版位），絕不叫 AI 畫中文字**——AI 畫的中文會亂碼，程式疊的每張都一模一樣，這才是可複製的紀律。

**標準產圖流程（與 `_codex` 雙圖流程整合，2026-07-08）**：
1. **組成分析**：改舊圖前先拆區塊（背景／商品／品牌區／文字層級／裝飾），有 PSD 先讀 PSD（在 `蝦皮賣場\` 商品資料夾）
2. **clean-base**：Codex 生成無字底圖（產品保真、留文字區）
3. **程式排字**：套版型（B2）疊合規文案 → 這就是正式版，不需再人工排字
4. **縮圖模擬驗收**：縮 240px 檢查列表可讀性，過了才給 Tony 拍板
5. **Loop 回寫**：任務後寫 `_codex\worklog\`（任務類型／路線／產出／問題／下次改進）

| 順位 | 圖 | 任務 |
|---|---|---|
| 1 | 暖底主圖（混血版） | 搜尋頁搶點擊：產品大＋黃標籤鉤子＋小副標，文字程式疊 |
| 2 | 內容物／成分圖 | 攤開陳列（flat lay），標籤化標注成分名稱 |
| 3 | 使用步驟圖 | 3 步驟圖解（放入浴缸→浸泡→取出），掃一眼就會用 |
| 4 | 生活情境圖 | 浴室／玄關場景，營造「用它的那個時刻」的嚮往 |
| 5 | 規格圖 | 尺寸、重量、數量，用線條標注 |
| 6 | 品牌理念圖 | 一句品牌話＋吉祥獸／logo，收尾建立記憶 |

## 三、圖上文案底線（法規是天花板也是地板）

權威來源：`C:\第二大腦\極光盾\品牌資料庫\文案內容室\language_boundary_rules.md`＋法規審核準則 v1.4。

**四不（圖上任何一個字都適用）**：不放大恐懼／不承諾結果／不神化產品／不寫醫療效果。

圖上標籤的安全寫法（UNIQLO 式短標籤 × 極光盾合規版）：

| ❌ 不能上圖 | ✅ 可以上圖 |
|---|---|
| 淨化負能量、有效防護 | 日常淨化儀式 |
| 安神、助眠、舒緩 | 睡前的安靜時刻（🟡慎用）、沐浴儀式 |
| 招財、招桃花、轉運 | 儀式感、狀態整理 |
| 100% 天然有效 | 成分：艾草・粗鹽・薰衣草（照實列） |
| 抵擋小人、斬斷煞氣 | 給自己保留空間的提醒 |

規則：**標籤只寫「事實」（成分、步驟、規格）和「儀式／氛圍」（情境、時刻），不寫任何「產品對人做了什麼」。**
拿不準的字，上圖前先對照法規準則 v1.4 的禁用詞分級。

**鐵律：廣告文案不寫人體接觸用途（2026-07-08 因「沐浴」失誤立此規）**
接觸皮膚的產品受化粧品法管制最嚴。禁寫的是「**產品接觸人體的使用方式**」：溶入沐浴／泡澡水、塗抹肌膚、貼膚配戴——即使包裝標籤印了也一樣：包裝是既有印刷,廣告是宣傳行為,不幫它放大。
**攜帶類用法可以寫**（口袋、包包、行李、車內、玄關）——產品封在袋裡沒有接觸人體,屬物品／空間用法（2026-07-08 誤判「口袋」後釐清）。
文案審查第五問（每次必問）：**「這句話有沒有讓產品接觸人體？」**——入水、上皮膚、貼膚戴的打回重寫;帶著走的不算。

## 四、圖庫版本管理（不留垃圾圖）

```
圖庫\
  01_討論中\   ← 新圖先進這裡，逐圖討論（不上 GitHub）
  02_定稿\     ← 討論通過才搬進來，依商品分資料夾（會同步 GitHub）
測試輸出\      ← 純技術測試，隨時可清（不上 GitHub）
```

- **命名規則**：`商品_用途_日期_v版本.png`，例：`沐浴包_主圖_0707_v2.png`
- **逐圖討論流程**：每張圖我會附三件事給 Tony 拍板——①風格說明（哪裡符合指南）②圖上文案（已過四不）③功能定位（圖組裡第幾張、任務是什麼）
- **淘汰即刪**：沒選上的直接刪掉，不留「以後再說」資料夾（Codex 在 `~\.codex\generated_images` 本來就有自動留底，要救隨時救得回來）
- 定稿圖每週一隨排程自動推上 GitHub 備份
